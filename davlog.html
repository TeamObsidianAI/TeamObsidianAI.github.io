April 2026 Pipeline

End-to-end pipeline working

The full automation loop is running: trend scan → script → voiceover → captions → upload. Top 3 trending topics from the YouTube Data API get turned into short-form videos and posted to TikTok via the Content Posting API.

  • Trend detection pulls the top 3 topics from YouTube's trending feed
  • Claude generates a 60-second script for each topic automatically
  • TTS voiceover rendered locally, captions auto-synced frame-by-frame
  • Videos assembled with ffmpeg and uploaded in sequence
April 2026 Auth

TikTok API integration

Getting the TikTok Content Posting API approved took the bulk of the setup time. The app is registered as a personal-use tool, so the privacy and terms pages are required by their developer portal review process.

  • OAuth flow stores the refresh token locally — never transmitted anywhere
  • Token refresh handled automatically before each upload session
  • Privacy policy and terms pages linked from the TikTok app dashboard
April 2026 Kickoff

Project started

The goal: a fully automated content pipeline that runs on a personal desktop machine, requires zero manual intervention per video, and posts to TikTok, YouTube, and Instagram simultaneously.

Starting constraints: no cloud spend, no subscriptions beyond APIs already in use, ship something real within a month.
